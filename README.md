# tic-tac-toe-java
A console based tic tac toe game built using java.

## Features
-Two players gameplay
-3x3 2d array board
-winner detection
-draw checker
-input validation
-improved board display

## Concepts used

-functions
-2D arrays
-conditional statements
-game logic
